SPRING BOOT APPLICATION
-----------------------

OVERVIEW
--------
THIS SOFTWARE ARCHITECTURAL PATTERN WILL BE USED TO SEPERATE AN APPLICATION INTO
ITS CLEAR RESPONSIBILITIES

OBJECTIVES COVERED BY PROJECT 
-----------------------------
1)CREATION OF A SPRING BOOT PROJECT FROM SCRATCH 

2)CREATION OF THE FIRST SPRING CONTROLLER

3)HANDLING OF THE HTTP GET REQUESTS

4)RETURNING OF HTML VIEWS AND PLAIN TEXT

5)MODEL VIEW CONTROLLER IMPLEMETATION 

DEPENDENCIES INVOVLED
---------------------
1)SPRING WEB

2)LOMBOK

3)THYMELEAF
